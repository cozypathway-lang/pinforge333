# PinForge — Pinterest Pin & CSV Generator

## Deploy in 3 clicks — no coding needed

### Step 1 — Push to GitHub
1. Go to github.com → sign up (free)
2. Click **+** → **New repository** → name it `pinforge` → Create
3. Click **uploading an existing file** → drag ALL files from the pinforge folder → Commit

### Step 2 — Deploy on Vercel
1. Go to vercel.com → sign up with GitHub
2. Click **Add New → Project** → select your `pinforge` repo → **Deploy**
3. Wait 2 minutes → your app is live!

### Step 3 — Add your API key inside the app
1. Open your live Vercel URL
2. Click **Settings** in the left sidebar
3. Choose your AI provider (Claude, ChatGPT, or Grok)
4. Paste your API key → Save
5. Done — start generating pins!

## Features
- Multi-AI: Claude, ChatGPT (GPT-4o), Grok, or any custom OpenAI-compatible API
- Multi-storage: Cloudinary, ImgBB, Amazon S3
- Projects system: work on multiple campaigns in parallel
- Pinterest-official CSV export format
- Editable table before export
- Schedule with time windows and pins-per-day control

## No Node.js needed
This project is deployed entirely via GitHub → Vercel. You never need to run anything locally.
